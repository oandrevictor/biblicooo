# Character role curation

`primaryRole` describes the character's defining function in the biblical
narrative. It is not a list of every office, relationship, or activity associated
with that person. Additional context belongs in `roleTags`.

Use this precedence when more than one role could apply:

1. A named office or position: monarch, political authority, prophet,
   priest/Levite, or apostle.
2. A sustained narrative function: judge/deliverer, disciple/follower,
   church leader/missionary, or opponent.
3. Patriarch/matriarch for foundational ancestors in a major biblical lineage.
4. Family member only when kinship is the main way the character is identified
   and no stronger role above applies.
5. Other only when none of the defined roles accurately describes the character.

People with the same given name must have distinct display names. A short name
may remain as an alias when it resolves to a single curated entity.
